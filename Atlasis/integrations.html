Vous ne trouvez pas
votre outil ?

Construisez des intégrations sur-mesure avec notre API REST complète, des webhooks et notre SDK open-source. La documentation est là pour vous guider.

atlasis.config.ts 
import { Atlasis } from '@atlasis/sdk'

const client = new Atlasis({
  apiKey: process.env.ATLASIS_KEY,
  region: 'eu-west-1',
})

// Connecter un outil personnalisé
await client.integrations.connect({
  name: 'Mon Outil',
  webhook: 'https://hooks.exemple.com',
  events: ['data.push', 'sync.done'],
})
