🌌 Atlasis — Premium AI Cloud Platform Template
[French Version below]
Atlasis is a modern, high-performance, and highly customizable template designed for AI SaaS platforms, cloud infrastructure, and data management. With its premium "Dark Mode" aesthetic, fluid animations, and modular structure, it is production-ready or suitable as a robust project foundation.

🚀 Key Features
Premium Design: Dark aesthetic with electric blue accents, featuring glassmorphism and micro-animations.
Fully Responsive: Optimized for desktop, tablet, and mobile devices.
Design System (CSS Variables): Customize the entire visual identity through a single file.
Modular Data (JS): Content management (Blog, Testimonials, Integrations) is isolated in simple constants for easy updates without touching HTML.
Advanced Animations: Canvas particles, magnetic hover effects, scroll reveals, and dynamic progress bars.

🛠️ Customization Guide

1. Colors & Identity
Modify :root variables in style.css to change the accent colors and backgrounds globally.
2. Content Updates
Simply open script.js or integrations.js and update the DATABASE constants to add your own articles or tools.

🌌 Atlasis — Template Premium pour Plateforme Cloud IA

Atlasis est un template moderne, performant et hautement personnalisable conçu pour les plateformes SaaS d'intelligence artificielle, d'infrastructure cloud et de gestion de données.

🚀 Fonctionnalités Clés

Design Premium : Esthétique sombre avec touches de bleu électrique, utilisant le "glassmorphism".
Entièrement Responsif : Adapté pour tous les écrans.
Système de Design : Personnalisation via variables CSS.
Base de Données Modulaire (JS) : Mise à jour du contenu simplifiée via des fichiers JavaScript dédiés.
Animations Avancées : Particules Canvas, scroll reveal et barres de progression.

🛠️ Guide de Personnalisation

1. Couleurs et Identité Visuelle
Ouvrez style.css pour modifier les variables globales :
code
CSS
:root {
  --accent-color: #0066FF; /* Votre couleur de marque */
  --bg-primary: #020205;   /* Fond principal */
}

2. Gestion du Contenu
Pour ajouter des articles ou des intégrations, modifiez simplement les tableaux BLOG_DATABASE dans script.js ou INTEGRATIONS_DATABASE dans integrations.js.

💻 Installation

Téléchargez le dossier.
Ouvrez index.html dans votre navigateur.
Note : Les pages de détail utilisent des paramètres URL (ex: ?id=slack) pour fonctionner dynamiquement.
